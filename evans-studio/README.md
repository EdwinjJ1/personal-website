# Evan Web

A standalone pixel open-world portfolio map built with Next.js and Phaser.

## Development

Generate the GPT Image 2 pixel asset pack first:

```bash
npm run generate:assets
```

Then run the development server:

```bash
npm run dev
```

Open the local URL printed by Next.js.

## Controls

- Desktop: WASD or arrow keys to move, E to interact, Esc to close panels.
- Mobile: on-screen controls.

## Assets

The art direction is defined in `docs/pixel-art-bible.md`. Runtime assets live under `public/media/pixel`.
